 📚 CSE Project Repository

> A comprehensive web-based platform for Computer Science Engineering students to showcase, manage, and collaborate on academic projects.

🎯 Overview

CSE Project Repository is a full-stack web application designed to empower Computer Science Engineering students to showcase their academic and personal projects. The platform provides an intuitive, professional interface for uploading projects, discovering others' work, and managing project details with ease. This project demonstrates industry-standard web development practices including database design, secure form validation, SQL injection prevention, XSS attack prevention, responsive web design, RESTful conventions, and professional code documentation.

 🌐 Tech Stack Breakdown

| Layer | Technology | Usage |
|-------|-----------|-------|
| **Backend** | PHP 7.4+ | 62.6% |
| **Frontend Styling** | CSS3 | 21.5% |
| **Frontend Interactive** | JavaScript ES6 | 15.9% |
| **Database** | MySQL 5.7+ | Core Data Storage |
| **Server** | Apache (WAMP/XAMPP) | Web Server |


 ✨ Core Features

 👤 Student Features
- ✅ **Upload Projects** - Easy-to-use form with comprehensive validation
- ✅ **Browse All Projects** - Grid view with real-time search and filtering
- ✅ **Search & Filter** - Find projects by title, student name, technology, or category
- ✅ **View Details** - Complete project information with metadata
- ✅ **Edit Projects** - Update project information anytime
- ✅ **Delete Projects** - Remove projects with safety checks
- ✅ **GitHub Integration** - Link to GitHub repositories directly
- ✅ **Responsive Design** - Seamless experience on mobile, tablet, desktop

 🛡️ Security Features
- ✅ **SQL Injection Prevention** - `mysqli_real_escape_string()` escaping
- ✅ **XSS Protection** - `htmlspecialchars()` sanitization
- ✅ **Type Safety** - Integer casting for numeric values
- ✅ **POST Validation** - Secure form submission handling
- ✅ **Client-Side Validation** - Real-time error feedback
- ✅ **Server-Side Validation** - Double-layer security

📊 Data Management
- ✅ **Project Statistics** - Total projects, students, categories
- ✅ **Timestamp Tracking** - Automatic upload date recording
- ✅ **Database Queries** - Optimized SELECT, INSERT, UPDATE, DELETE
- ✅ **Category Management** - Dynamic category filtering
- ✅ **Sample Data** - Pre-populated demo projects
